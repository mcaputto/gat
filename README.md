# gat

Create new [gatsby-starter-blog](https://github.com/gatsbyjs/gatsby-starter-blog)
posts.

```sh
path/to/gat [-h] title
```

## Examples

```sh
$ ./gat foobar
Created src/pages/2017-11-05-foobar/index.md
```

```sh
$ cat src/pages/2017-11-05-foobar/index.md
---
title: foobar
date: "2017-11-05 14:39:59.282014"
path: "/foobar/"
---


```
